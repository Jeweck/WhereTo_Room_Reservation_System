# **App Name**: AcademiaReserve

## Core Features:

- Secure User Authentication: Separate login pages for administrative staff and students/faculty, leading to distinct dashboards based on role.
- Admin Facility Management: Admins can create, edit, view, and delete facilities including details like name, capacity, available equipment, and purpose.
- User Facility Browsing & Search: Users can view a list of available facilities, search by keywords, and filter by attributes like capacity or equipment.
- Booking Creation with Conflict Resolution: Users can submit reservation requests for facilities; the system automatically checks for and prevents time conflicts for chosen dates and times.
- Personalized Booking Dashboard: Users and Admins view their upcoming and past bookings in a clean dashboard interface, providing a clear overview.
- Smart Facility Recommendation Tool: An AI tool that suggests the most suitable available facility based on user-inputted requirements (e.g., meeting size, required presentation tools, study room).

## Style Guidelines:

- Primary color: Deep Prussian Blue (#2B2C6E) for a sense of trust and institutional rigor. Hue is approximately 240 degrees in HSL.
- Background color: Soft, desaturated Lavender Mist (#EEF0FB) providing a calm and focused canvas. It visibly belongs to the same hue as the primary but is much lighter and less saturated.
- Accent color: Vibrant Teal (#00B0CB) for highlighting interactive elements and calls to action, chosen to provide energetic contrast while harmonizing with the primary blue.
- Body and headline font: 'Inter', a clean and modern grotesque sans-serif font, chosen for its excellent readability and neutral, professional aesthetic suited for both display and extended text.
- Use clear, concise line icons that are easily recognizable for navigation and interactive elements, matching the streamlined interface.
- Implement a two-column layout with a fixed, dark navigation sidebar on the left and a main content area for information and interactions, as suggested by the reference images.
- Subtle and swift animations for state changes, form submissions, and page transitions to provide fluid user feedback without distracting from tasks.
